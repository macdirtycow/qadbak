We build Qadbak and ship the iOS app that talks to it.

Inveil is the company behind Qadbak — an open-core hosting panel you install on your own VPS — and the native iPhone/iPad app operators use to manage that panel on the road.

Qadbak panel

Domains, mail, DNS, TLS, backups, app store, API, and security tools on Debian 12 or Ubuntu LTS. Source on GitHub. Premium modules (client logins, webmail, offsite backups) via license.inveil.dev.

iOS app (TestFlight)

Two modes: connect to your Qadbak panel URL for full hosting control, or add an existing Linux server with the Qadbak agent — including read-only links to HestiaCP, Coolify, and CasaOS. Request a beta seat at support@inveil.net.
