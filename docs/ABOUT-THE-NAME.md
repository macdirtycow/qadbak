# About the name Qadbak

The panel is **not** affiliated with the former investment vehicle. The name is deliberately reused with a different meaning: **real infrastructure instead of a hollow pitch.**

## Historical note (2009)

[Qadbak Investments](https://en.wikipedia.org/wiki/Qadbak_Investments) appeared around 2009 as a mysterious consortium that briefly linked itself to English football club **Notts County** (including appointing Sven-Göran Eriksson) and reportedly neared a deal for the **BMW Sauber** Formula 1 team. Press coverage treated it as a major incoming backer; investigations later described it as a structure with little substance behind the branding. See also [Reuters (2009)](https://www.reuters.com/article/sports/bmw-sell-f1-team-to-swiss-based-qadbak-idUSTRE58E675/).

In finance and sport, the name became associated with **vapourware** — big announcements, thin reality.

## This project (2020s)

| Then (2009) | Now (Qadbak panel) |
|-------------|-------------------|
| “Investment” with unclear backers | Panel code you run on **your** VPS |
| Promised clubs and F1 teams | Delivers **hosting**: domains, mail, DNS, SSL |
| Name avoided in serious finance | Name reclaimed for **hosting control** |

**Qadbak** here means: the **front door** on ports 80/443 for your server — a self-contained control panel, not a fork of anything else.
